Pagina principal servicios tiendas gastronomia Quienes Somos tegnologias

Definición del Proyecto:

Este proyecto ha sido creado utilizando HTML y CSS. El propósito es construir una página web para un centro comercial, donde se utilizan divs para estructurar el contenido y CSS para dar formato y diseño visual.

Herramientas Utilizadas:

HTML:

Utilizado para la creación de la estructura básica de la página, incluyendo las secciones como encabezado, menú de navegación, contenido y pie de página.

CSS:

Usado para diseñar la presentación visual de la página. Esto incluye el manejo de colores, tamaños, márgenes, alineación, y efectos visuales como los gradientes y transiciones. Además, se ha aplicado la propiedad Flexbox para organizar los elementos de manera adaptable a diferentes tamaños de pantalla.

Divs:

Se utilizan para crear contenedores dentro de la página, permitiendo una organización clara del contenido. Cada sección de la página está contenida dentro de un div, lo que facilita la estructuración y el diseño responsivo.

Propósito:

El objetivo es desarrollar un sitio web atractivo, fácil de usar y optimizado para dispositivos móviles, que muestre las diversas secciones del centro comercial, tales como tiendas, gastronomía, entretenimiento y más, utilizando herramientas y tecnologías básicas de diseño web.
